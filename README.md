# Image Processing and Computer Vision
## Semister 6

## Lab Index

| Lab No | Title                                   | Description                                                                 |
|-------|------------------------------------------|-----------------------------------------------------------------------------|
| LAB1  | Basic Image Processing                    | Introduction to image processing concepts and different types of image processing techniques. |
| LAB2  | Geometric Primitives                      | Implementation of image resizing, rotation, translation, and basic image alterations. |
| LAB3  | Edge, Line, & Corner Detection            | Detection of edges, lines, and corners using multiple algorithms such as Sobel, Canny, Hough Transform, and Harris Corner Detection. |
| LAB4  | Image Histogram & Histogram Equalization  | Computation of image histograms and enhancement of image contrast using histogram equalization techniques. |
| LAB5  | Skin Colour Detection                     | Detection and segmentation of human skin regions in images using color space transformations and thresholding techniques. |
| LAB6  | Image Warping & Transformation Estimation | Implementation of geometric transformations and image warping using transformation matrices and coordinate mapping. |
| LAB7  | Motion Tracking                           | Tracking motion in video sequences by analyzing frame differences and detecting moving objects over time. |
| LAB8  | Object Detection using YOLOv8             | Implementation of real-time object detection using the YOLOv8 model on images and videos. |
| LAB9  | Depth Estimation                          | Estimating depth information from stereo images using left and right image pairs. |
| LAB10 | Feature Matching using RANSAC             | Matching features between two images and estimating transformations using RANSAC for robust matching. |
