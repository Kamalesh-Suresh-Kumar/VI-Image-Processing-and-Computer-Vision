# Image Processing and Computer Vision
## Semister 6

## Lab Index

| Lab No | Title                                       | Description                                                                                                                               |
| ------ | ------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| LAB1   | Basic Image Processing                      | Introduction to image processing concepts and different types of image processing techniques.                                             |
| LAB2   | Geometric Primitives                        | Implementation of image resizing, rotation, translation, and basic image alterations.                                                     |
| LAB3   | Edge, Line, & Corner Detection              | Detection of edges, lines, and corners using six different algorithms such as Sobel, Canny, Hough Transform, and Harris Corner Detection. |
| LAB4   | Image Histogram & Histogram Equalization    | Computation of image histograms and enhancement of image contrast using histogram equalization techniques.                                |
| LAB5   | Skin Colour Detection                       | Detection and segmentation of human skin regions in images using color space transformations and thresholding techniques.                 |
| LAB6   | Image Warping and Transformation Estimation | Implementation of geometric transformations and image warping using transformation matrices and coordinate mapping.                       |
| LABn   | To be Updated                               | Additional computer vision experiments and advanced image processing techniques to be added in future labs.                               |
